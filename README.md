# ♠️ Blackjack Game

A simple browser-based Blackjack game built with JavaScript, HTML, and CSS. Try your luck against the dealer in this classic card game!

## 🕹️ Features
- Playable Blackjack game logic (Hit, Stand, Win/Lose)
- Score tracking for player and dealer
- Basic card animations
- Simple and responsive UI
- Clean code and modular structure

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (vanilla)
